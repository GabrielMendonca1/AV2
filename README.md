# AV2 

Descrição breve do projeto.

## Tecnologias Utilizadas

- React Native
- Rapid API
- Expo Router

## Pré-requisitos

- Node.js (versão LTS)
- Git

## Instalação

git clone https://github.com/GabrielMendonca1/AV2

**Nota:** Não use `npm update`.

agora use: 

`npm install`

e depois inicie o projeto com: 

`npx expo start`
